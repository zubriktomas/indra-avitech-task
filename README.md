# Indra Avitech Task

### Dependencies
Java 8, Maven

### Installation
```sh
git clone https://github.com/zubriktomas/indra-avitech-task.git
mvn package
```

### Usage
```
cd target
java -jar avitech-app-1.0.jar input_file [key]
```

### Example
Immediate output:
```
java -jar avitech-app-1.0.jar ./../input.txt Brano
```

Interactive:
```
java -jar avitech-app-1.0.jar ./../input.txt
```
